# Example typescript CrossCode mod with [gleam](https://gleam.run/)

## Building

You need `gleam` installed

```bash
git clone https://github.com/krypciak/cc-ts-glem-template
cd cc-ts-glem-template
pnpm install
pnpm run start
# this should return no errors (hopefully)
npx tsc
```
